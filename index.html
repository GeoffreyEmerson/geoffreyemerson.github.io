About

Geoffrey Emerson is a full stack web developer currently residing in Portland, Oregon. He started programming for fun at a very young age, and in 2015 decided to make the switch from hobbyist to professional. Geoff has experience in JavaScript, Ruby on Rails, and PHP, and enjoys learning new languages.

Over the years he has also studied Basic, Pascal, C, C++, C# for Xbox, and Objective-C for iOS. While he took some programming classes for fun in college, he majored in Theatre Arts with an emphasis Acting and Musical Theatre. But what he really wants to do is direct!

AssetDB

This is a simple browser based inventory tracking system I created circa 2006. It uses PHP and a PostgreSQL database. To log into this demo site, simply enter any username. No need for a password. It is completely unpolished, and was only built out as far as we needed it when it was created.

  • Demo: AssetDB (Hosted on Heroku)
  • Code base: Github Repo
  • Date: 2006
  • Technologies: HTML, CSS, PHP, PostgreSQL
